# Todo List Web Application

This is a simple to-do list web application.

## Features
- Add new todo items.
- Mark todo items as completed.
- Delete completed todo items.

## How to Use
1. Type your todo item in the input field.
2. Click "Add" to add the todo item to the list.
3. Click on a todo item to mark it as completed.
4. Click again on a completed todo item to unmark it.
5. To delete a completed todo item, simply click on it.

Enjoy organizing your tasks with this simple tool!
